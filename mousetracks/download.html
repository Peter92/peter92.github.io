Mouse Tracks (WIP)

Below you will find all the official download links and mirrors. You can also grab the source code from github if you have a working Python installation, but it is not guarenteed to be bug free.

The graphical interface will come later, so as a quick tutorial to get it up and running, load StartTracking.exe to activate the tracking, and GenerateImages.exe when you want to render the tracks (leave it running for a short while first).

If you would like to hear about future updates, sign up here (it's quick and I promise there won't be any spam) and you'll get an email when I release a new version. Alternatively follow the page on Facebook.
